## What's New in v3.0

The v3.0 methodology fuses the original dietary-guide distillation methodology with the **book-to-skill** framework (by virgiliojr94), creating a unified approach for distilling both official guidelines and books into structured AI Skills.

Key additions:
- Dual knowledge architecture (KPK + Framework)
- 4-mode workflow (Full / Analyze / Generate from analysis / Incremental update)
- Token cost estimation & REPL-style large document access
- Chapter summary generation + supporting files (glossary, patterns, cheatsheet)
- Incremental update workflow for existing skills
- 11 quality rules with cross-source attribution
